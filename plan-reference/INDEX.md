## How to Use This Folder

### For Developers Starting Work

1. Read [STATUS.md](./STATUS.md) for current project status
2. Check relevant design docs (e.g., [navigation-system-design.md](./navigation-system-design.md))
3. Follow the roadmap (e.g., [navigation-system-roadmap.md](./navigation-system-roadmap.md))
4. Update STATUS.md as tasks are completed

### For Planning New Features

1. Review existing [strategy.md](./strategy.md)
2. Check [ui-ux-design.md](./ui-ux-design.md) for design patterns
3. Create new design doc in this folder
4. Update [ROADMAP.md](./ROADMAP.md) with timeline
5. Update STATUS.md with new tasks

### For Documentation Updates

- User-facing docs: Update files in `../docs/`
- Developer docs: Update files in this folder
- Always update STATUS.md when completing tasks
